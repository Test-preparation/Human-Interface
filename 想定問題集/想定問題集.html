計算1（MHP）

2つ並んで表示された画像を見比べて、同じ絵柄かどうかを判断し、同じならスペースキーを押す作業を、モデルヒューマンプロセッサ(MHP)で見積もれ。作業は次の5ステップに分解できるものとする。

  1. 左の画像を知覚する(知覚)
  2. 右の画像を知覚する(知覚)
  3. 2つの画像を比較する(認知)
  4. 同一であると判断する(認知)
  5. スペースキーを押下する(運動)

各プロセッサの処理時間は、知覚プロセッサ=100ms、認知プロセッサ=70ms、運動プロセッサ=70msとする。作業達成時間Tを求めよ。

解答

直列加算モデルなので、各ステップの処理時間を単純に足す。

  • 知覚ステップ:2回(ステップ1・2) → 100ms × 2 = 200ms
  • 認知ステップ:2回(ステップ3・4) → 70ms × 2 = 140ms
  • 運動ステップ:1回(ステップ5) → 70ms × 1 = 70ms

T = 200 + 140 + 70 = 410ms

計算2（視力・視角）

視力1.5のユーザが、0.25m(=250mm)の距離でディスプレイを観察する。ピクセルが識別不可能になる限界の解像度[ppi]を求めよ。視力1.0は視角1/60°(1分)の物体を識別できることと定義され、視力と識別可能な最小視角は反比例する。π≈3.14、1インチ=25.4mmとし、tan(x)≈x(弧度法)の近似を用いてよい。

解答
  • 視力1.5の識別可能な最小視角:θ = (1/60)/1.5 = 1/90°
  • 弧度法に変換:θ(rad) = (1/90) × (π/180) = (1/90) × (3.14/180) = 3.14 / 16200 ≈ 0.0001939 rad
  • 限界サイズ(tan(x)≈x の近似で w ≈ D × θ(rad)):

w ≈ 250mm × 0.0001939 ≈ 0.04846mm

  • 解像度:ppi = 25.4 ÷ w = 25.4 ÷ 0.04846 ≈ 約524ppi

(参考:近似せず w = 2 × D × tan(θ/2) で計算してもほぼ同値。)

計算3（フィッツの法則）

あるポインティングデバイスについて、フィッツの法則を MT = a + b × log2(2D / W) の形で用いる。定数を a=50ms、b=150ms/bit とし、スタート地点からターゲット中心までの距離D=8cm、ターゲットの幅W=1cmとする。運動時間MTを求めよ。

解答
  • 作業難度:ID = log2(2D/W) = log2(2×8/1) = log2(16) = 4 bit
  • 運動時間:MT = a + b×ID = 50 + 150×4 = 50 + 600 = 650ms
計算4（KLM）

一般ユーザが、マウスで画面上部の「ファイル」メニューを開き、その中の「印刷」項目を選んでクリックする操作を、キーストロークレベルモデル(KLM)で見積もれ。操作開始時、両手はキーボード上にあるものとする。操作子の所要時間は K=0.20s、P=1.10s、H=0.40s、M=1.35s とし、Mの挿入・省略ルール(文字以外のK/指令のPの前にMを置く/P直後のKのMは省略)に従うこと。所要時間Tを求めよ。

解答

操作を分解し、Mの挿入・省略ルールを適用する。

  1. マウスに手を移動:H
  2. 「ファイル」メニューにカーソル移動:M+P(新しい判断が必要な指令のPなのでMを置く)
  3. クリック:(M省略)+K(直前のPから予測できるのでM省略)
  4. 「印刷」項目にカーソル移動:M+P(新しい判断なのでM)
  5. クリック:(M省略)+K(直前のPから予測できるのでM省略)

操作子の集計:H×1、P×2、K×2、M×2

T = H + 2P + 2K + 2M

= 0.40 + 2×1.10 + 2×0.20 + 2×1.35

= 0.40 + 2.20 + 0.40 + 2.70

= 5.70s
